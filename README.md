# sql-select-fundamentals
Ejercicios básicos de SQL enfocados en el uso de SELECT para la exploración y consulta de datos.

## Objetivo

Practicar los fundamentos de consultas SQL utilizando la sentencia SELECT para explorar y recuperar información de una base de datos.

## Contenido

### Consulta 1 - Exploración general

Uso de SELECT * para visualizar todas las columnas de la tabla sales y comprender su estructura.

### Consulta 2 - Selección específica

Obtención de las columnas customer_id, product_id y total_amount para responder a necesidades del área financiera.

### Consulta 3 - Alias de columnas

Renombrado de columnas mediante AS para mejorar la legibilidad de los resultados para usuarios de negocio.

## Conceptos aplicados

* SELECT
* SELECT *
* Selección de columnas específicas
* Alias con AS
* Documentación mediante comentarios SQL
